> **StealthAimer** — это аппаратный аим-ассистент нового поколения (2026 Edition). Система использует нейросети YOLOv11 с оптимизацией TensorRT и внешние микроконтроллеры для полной имитации физической мыши.

---

## 🧠 Как это работает

Система построена на принципе разделения вычислений и действий, что делает её невидимой для софтверных античитов:

1.  **Захват экрана**: Библиотека `dxcam` вырезает центральную область (FOV) с минимальной задержкой.
2.  **Детекция целей**: YOLOv11, работающая на тензорных ядрах **RTX 20+**, находит противников быстрее чем за 1 мс.
3.  **Адаптивный хитбокс**: Точка прицеливания динамически смещается в зависимости от размера цели (ближе — точнее, дальше — больше разброс).
4.  **Humanized Logic**: Применяется плавное сглаживание с джиттером и случайные задержки реакции (50–250 мс) для обхода серверной статистики.
5.  **Hardware Mouse**: Координаты уходят по Serial на **Arduino Leonardo**, которая аппаратно двигает курсор.

## 📦 Готовый исполняемый файл (Release)

Для тех, кто не хочет настраивать окружение Python, доступна сборка в один клик:

1.  Скачайте архив из [Releases](https://github.com/a?shokakjehochetsyapisechkudjenskiuyv1.0Stable/releases).
2.  **Прошивка**: Во вкладке "Прошивка" выберите COM-порт вашей Arduino и нажмите "Прошить".
3.  **Модель**: Поместите ваш `.engine` файл в папку с программой.
4.  **Запуск**: Нажмите **F1** для активации системы в игре.
